Responsive CSS
==============

Common css classes for responsive design (adopted from twitter bootstrap).

Screen Definition
-----------------

* Large Screen
    > min-width: 1200px

* Tablet Landscape <br />
    > max-width: 1199px<br />
    > min-width: 980px

* Tablet Portrait<br />
    > max-width: 979px<br />
    > min-width: 768px

* Phone Landscape<br />
    > max-width: 767px<br />
    > min-width: 481px<br />

* Phone Portrait<br />
    > max-width: 480px

Available Classes
-----------------

    .visible-large-screen
    .visible-tablet-landscape
    .visible-tablet-portrait
    .visible-phone-landscape
    .visible-phone-portrait
    
    .visible-below-large-screen
    .visible-below-tablet-landscape
    .visible-below-tablet-portrait

    .hidden-large-screen
    .hidden-tablet-landscape
    .hidden-tablet-portrait
    .hidden-phone-landscape
    .hidden-phone-portrait

    .hidden-below-large-screen
    .hidden-below-tablet-landscape
    .hidden-below-tablet-portrait
